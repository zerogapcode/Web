Instalación en 4 pasos

1

Descarga la App

Haz clic en el botón de abajo para descargar el archivo .dmg. Una vez descargado, ábrelo y arrastra AXIOM a tu carpeta de Aplicaciones.

Build aarch64 · Apple Silicon M1 / M2 / M3 / M4
2

Aviso de seguridad de macOS

Como AXIOM está en fase Beta y aún no tiene firma oficial de Apple, macOS puede mostrar el mensaje "la aplicación está dañada". Esto es completamente normal en apps en desarrollo.
3

Autoriza la aplicación

Abre Terminal (búscala en Spotlight), copia el comando y presiona Enter:

xattr -cr /Applications/AXIOM.app
4

¡Todo listo!

Cierra la Terminal y abre AXIOM desde Launchpad o tu carpeta de Aplicaciones. Asegúrate de tener Ollama corriendo en tu Mac.

Preguntas frecuentes

¿Necesito pagar para usar AXIOM?
No. AXIOM es completamente gratuito en su versión beta. Usamos Ollama con el modelo Qwen2.5:7b, que también es de uso libre. No hay suscripciones.
¿Funciona con Intel Mac?
La versión actual está compilada para Apple Silicon (M1/M2/M3/M4). Un build para Intel x86-64 está en el roadmap para versiones futuras.
¿Mis datos y correos son privados?
Absolutamente. La IA corre de forma local en tu Mac mediante Ollama. Ningún dato sale de tu equipo. Tu correo se lee y procesa únicamente en la memoria de tu computadora.
¿Necesito saber programar para usarlo?
Para nada. AXIOM está diseñado para usuarios no técnicos. Si puedes escribir un mensaje de WhatsApp, puedes usar AXIOM.
¿Qué pasa si AXIOM intenta hacer algo que no quiero?
Para acciones sensibles (como enviar correos o eliminar archivos), AXIOM siempre te pide confirmación antes de proceder. Tú tienes el control final en todo momento.
